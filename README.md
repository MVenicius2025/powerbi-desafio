# Power BI – Desafio de Projeto DIO

Este repositório contém a entrega do **Desafio de Projeto Power BI – DIO**, onde foram aplicados conceitos de análise e visualização de dados utilizando o dataset `Financial Sample.xlsx`.

---

## 🎯 Objetivo
Replicar duas páginas criadas durante o curso e desenvolver uma **terceira página** com novos visuais, explorando recursos de mapas e gráficos de pizza no Power BI.

---

## 📂 Estrutura do Repositório
📂 powerbi-desafio
├── README.md
├── dataset/Financial Sample.xlsx
└── desafio_powerbi.pbix

---

## 📊 Páginas do Relatório

### Página 1
- Gráfico de pizza: **Vendas por Produto**  
- Gráfico de linha: **Preço médio de venda por Produto**  
- Gráfico de barras: **Vendas por Segmento ao longo do tempo**

### Página 2
- KPIs: **Soma de Sales, Profit e Units Sold**  
- Gráfico de barras: **Vendas por País**  
- Gráfico de linha: **Lucro por Ano/Mês**  
- Filtros interativos: **Ano, Trimestre, Mês, Segmento**

### Página 3 (Desafio)
- **Mapa 1:** Soma de *Sales* e *Units Sold* por País  
- **Mapa 2:** Soma de *Profit* por País  
- **Pizza:** Lucro por Segmento

## 🚀 Publicação
